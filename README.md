## ðŸ— Project Scope & Requirements

As per the course guidelines, our project is built around a real-world **SSM-registered company**. We are strictly adhering to the following technical requirements:

* **Database Scale**: Minimum of **20 entities**, each with at least **10 attributes**.
* **Modeling**: An **Enhanced Entity-Relationship (EER)** diagram in **3rd Normal Form (3NF)**.
* **Key Constraints**: Each entity features **composite primary keys** (2 attributes).
* **Hierarchies**: Superclass entities containing at least 2 common non-primary key attributes.
* **Validation**: Each entity is populated with a minimum of **10 rows of sample data**.
